Introduction to JSON (JavaScript Object Notation):

The nested dictionary is a representation of JSON.
JSON uses key-value pairs and supports various data types, including strings, numbers, objects, arrays, booleans, and null.
An online JSON editor is recommended for practicing and visualizing JSON data.

Introduction to YAML (YAML Ain't Markup Language):

YAML is a human-readable data serialization format and is commonly used for configuration files.
YAML uses indentation for structure.
Recommends an online YAML editor for practice.
yaml

skill: DevOps
year: 2023
tech:
  cloud: AWS
  containers: Docker
  ci_cd: Jenkins
  gitops:
    - GitLab
    - Argo CD
    - Tecton
Comparing JSON and YAML:

Highlights the similarities between JSON and YAML.
Both use key-value pairs and support nested structures.
